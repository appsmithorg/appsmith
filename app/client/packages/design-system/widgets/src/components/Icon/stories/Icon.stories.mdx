# Sizes

The Icon component comes with three sizes

- small — 1rem = 16px
- medium — 1.5rem = 24px
- large — 2rem = 32px

<Story name="Size">
  <div style={{ display: "flex", gap: "10px", flexWrap: "wrap" }}>
    {["small", "medium", "large"].map((size) => (
      <div key={size}>
        <Icon size={size} name="home" />
      </div>
    ))}
  </div>
</Story>
