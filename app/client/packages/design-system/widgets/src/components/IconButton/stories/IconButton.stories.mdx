## Semantic

`IconButton` component has 3 visual style variants and 5 semantic color options, similar to `Button` component.

<Story name="Semantic">
  <Flex gap="spacing-2" wrap="wrap">
    {["filled", "outlined", "ghost"].map((variant) =>
      ["accent", "neutral", "positive", "negative", "warning"].map((color) => (
        <div key={`${variant}-${color}`}>
          <IconButton variant={variant} color={color} icon="plus" />
        </div>
      )),
    )}
  </Flex>
</Story>

## Size

`IconButton` component has 3 size options. `small`, `medium`, `large`.

<Story name="Size">
  <Flex gap="spacing-2">
    {Object.keys(SIZES)
      .filter((size) => !["large"].includes(size))
      .map((size) => (
        <div key={size}>
          <IconButton size={size} icon="plus" />
        </div>
      ))}
  </Flex>
</Story>
